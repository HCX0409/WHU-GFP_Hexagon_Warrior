### 1.1 模型训练决策分支

```
输入: GFP_data.xlsx (141,572条)
  │
  ├─ 数据质量诊断
  │   ├─ 亮度分布分析 → 发现双峰分布(死蛋白+正常)
  │   ├─ 类型统计 → 4种GFP, WT亮度差异大
  │   └─ 决策: delta策略(实验亮度-WT亮度)消除系统偏差
  │
  ├─ V4基线探索
  │   ├─ 方法: 传统ML(XGBoost+Stacking)
  │   ├─ 结果: R²=0.46~0.84, 高亮度区偏差大
  │   └─ 决策: 淘汰, 转向蛋白质语言模型
  │
  ├─ V5 LoRA全量训练
  │   ├─ 架构: ESM-2(650M) + LoRA(r=32) + TypeEmbed
  │   ├─ 数据: 全量122k
  │   ├─ 结果: R²=0.44~0.82
  │   └─ 决策: 保留为基线(覆盖广, 识别死蛋白)
  │
  ├─ V7 Frozen MLP
  │   ├─ 架构: 冻结ESM-2嵌入 + 独立MLP(1024→128)
  │   ├─ 数据: 72k优质(分层采样)
  │   ├─ 结果: R²=0.921
  │   └─ 决策: 用作初筛(体量小, 推理快)
  │
  └─ V9 LoRA+深层MLP
      ├─ 架构: ESM-2 LoRA(r=16) + V7深层MLP头
      ├─ 数据: 72k优质
      ├─ 结果: R²=0.946
      └─ 决策: 主力模型(权重50%)
```

### 1.2 序列生成与筛选决策分支

```
输入: 4种GFP × Top50父本 = 200条
  │
  ├─ Phase 2: 分层突变
  │   ├─ 每条500变体 × 5轮 = 500,000条原始
  │   ├─ V7初筛(阈值: 各类型独立设定)
  │   │   ├─ avGFP: >3.0
  │   │   ├─ amacGFP: >3.5
  │   │   ├─ cgreGFP: >4.0
  │   │   └─ ppluGFP: >3.8
  │   ├─ V9+V5共识投票(≥2票通过)
  │   └─ 输出: 6,626条候选
  │
  ├─ Phase 3: ESM3-ddG热稳定性
  │   ├─ 逐位点ddG预测 → 求和
  │   ├─ 过滤: total_ddG > +3.0 排除(严重去稳定)
  │   └─ 输出: ddG打分CSV
  │
  └─ Phase 4: 综合筛选
      ├─ 加权: 0.50×V9 + 0.35×V7 + 0.15×V5
      ├─ 排序: weighted_avg降序
      ├─ 多样性: 每类型至少1条
      └─ 输出: Top 20
```

### 1.3 最终候选选择逻辑

```
输入: 4个数据源 (39,020条去重后)
  │
  ├─ 数据源合并
  │   ├─ Phase2.5 full_scored (已有三模型打分)
  │   ├─ Phase2 crossval (交叉验证)
  │   ├─ 探索性扫描 (早期80k→32k)
  │   └─ Phase4 Top20
  │
  ├─ 统一三模型打分 → weighted_avg
  │
  ├─ ESM3-ddG复打分
  │
  └─ 选择策略
      ├─ 排序: weighted_avg降序
      ├─ 每GFP类型至少1条
      ├─ AlphaFold 3结构审核
      │   ├─ beta-barrel完整性
      │   └─ 发色团pLDDT ≥ 70
      └─ 输出: 5条最终候选
```

## 三、Agent 关键决策记录

| 决策点 | 问题 | Agent分析 | 决策 |
|--------|------|-----------|------|
| 数据策略 | 4种GFP的WT亮度差异大 | delta=实验亮度-WT亮度可消除系统偏差 | 采用delta空间训练 |
| V4→V5 | 传统ML天花板 | ESM-2语言模型捕获进化信息 | 切换LoRA微调 |
| V5→V7 | 全量122k有噪声 | 分层采样72k优质数据 | 72k聚焦训练 |
| V7架构 | LoRA+梯度检查点不稳定 | 冻结ESM-2只训练MLP头 | Frozen MLP策略 |
| V9融合 | V7快但不微调 | LoRA微调+深层MLP头 | V9=V5精度+V7深度 |
| 筛选策略 | 单模型不可靠 | 三模型加权+共识投票 | 0.50V9+0.35V7+0.15V5 |
| ddG角色 | 亮度与稳定性弱相关(r=0.175) | ddG仅辅助过滤,不做主排序 | 阈值+3.0硬过滤 |
| 候选多样性 | 全部偏向cgreGFP | 每类型至少保证1条 | 类型约束选择 |
